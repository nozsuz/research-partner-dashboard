# Research Partner Dashboard

研究者検索・マッチングダッシュボード

## 機能

- 🔍 研究者検索（セマンティック検索・キーワード検索）
- 🧠 AIによる研究者分析
- 📊 マイプロジェクト管理
- 📄 契約・知財サポート
- 💰 資金調達サポート

## ファイル構成

- `index.html` - メインダッシュボード
- `config.js` - 設定ファイル
- `api-client.js` - API通信クライアント
- `test-button.html` - テスト用ページ
- `researcher-analysis.html` - 研究者分析ページ
- `temp-project.html` - 仮プロジェクト管理ページ
- `project-create.html` - プロジェクト作成ページ

## 使い方

1. `index.html` をブラウザで開く
2. 研究者検索タブで検索を実行
3. マイプロジェクトタブでプロジェクトを管理

## テスト

- `test-button.html` でテスト用プロジェクトを作成可能
- ブラウザの開発者ツールでコンソールログを確認
