🔧 API接続テストツール

APIサーバーURL設定

※ Railway管理画面のNetworking > Public Networkingで確認できます

接続テスト

よくあるRailway URL例

  • web-production-xxxx.up.railway.app
  • researcher-search-app-production.up.railway.app
  • your-project-name-production.up.railway.app

注意: URLの最後に / は含めないでください

トラブルシューティング

  • CORS エラー: ブラウザの開発者ツールでCORSエラーを確認
  • タイムアウト: サーバーの起動に時間がかかっている可能性
  • 404 エラー: URLが間違っているか、サーバーが停止中
  • 500 エラー: サーバー内部エラー（環境変数やGCP認証の問題）
